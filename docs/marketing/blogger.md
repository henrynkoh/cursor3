**Opening paragraph**

Greater Seattle 지역의 초기 스타트업 팀을 위한 **1주 집중 프로그램**을 소개합니다. 이름하여 **Seattle Cursor Vibe Week**. 목표는 단순히 “AI로 코드를 짠다”가 아니라, **Cursor 3**를 실제 제품 개발 파이프라인에 녹이는 것입니다 — 새 UI, 내장 브라우저로 디자인-투-코드, 서브에이전트 병렬 작업, 그리고 **GitHub까지 한 흐름으로 배포**합니다.

**What participants do (bullet list)**

- Day 1: 온보딩 & Cursor 3 기본 — 에이전트 패널, 터미널, 멀티 에이전트 가시성  
- Day 2: 브라우저 도구 — 랜딩/대시보드에서 요소 선택 후 프롬프트로 UI 반복  
- Day 3: 서브에이전트 — 보안·접근성·리팩터 병렬  
- Day 4: GitHub — 브랜치, PR, 리뷰  
- Day 5: 비용·생산성 — Cursor vs 대안 도구, 팀 정책 초안  
- Day 6: 종일 스프린트  
- Day 7: 데모데이 & 회고
